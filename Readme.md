## Commands

### Server

```bash
$ npm install
```

Installs all the required packages for backend

```bash
$ npm start
```

Starts the server using nodemon

## Client

```bash
$ npm install
```

Installs all the required packages for frontend

```bash
$ npm start
```

Runs frontend on localhost(React App)

```bash
$ npm run build
```

Creates an optimized production build
